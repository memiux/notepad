# 📝 Notepad [![Rating](https://img.shields.io/badge/Rating-★★★★☆-yellow.svg)](https://github.com/memiux/notepad)

A barebones notepad app using Sinatra and JavaScript, [try demo](https://notepad-demo.herokuapp.com) or

[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy?template=https://github.com/memiux/notepad)

---

**DEVELOPMENT**

	git clone git@github.com:memiux/notepad.git
	make build
	make
	http://localhost:8000/
